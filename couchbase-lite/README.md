# Module couchbase-lite

## Couchbase Lite Community Edition

Kotbase core Couchbase Lite Community Edition library

### Installation

```kotlin
kotlin {
    sourceSets {
        commonMain {
            dependencies {
                implementation("dev.kotbase:couchbase-lite:3.0.15-1.0.1")
            }
        }
    }
}
```

```kotlin
repositories {
    mavenCentral()
}
```
